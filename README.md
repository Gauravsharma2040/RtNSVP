# Real-Time Network Threat Visualization

A real-time network traffic monitoring system built with:
- FastAPI + WebSockets (backend)
- Vite + Vanilla JS + Leaflet (frontend)
- Streaming metrics & geospatial visualization

Features:
- Live event streaming
- DDoS vs normal traffic classification
- Real-time metrics aggregation
- World map visualization of threats

Planned:
- Integration with public intrusion detection datasets (CICIDS)
- Alerting and aggregation logic
